Benchmark Methodology

Test Environment

  • Build: Rust release mode with LTO
  • Features: --features benchmarks
  • RNG: Deterministic for reproducibility, CSPRNG for security tests
  • Iterations: Multiple runs with statistical aggregation

Comparison Notes

  • Traditional FHE comparisons based on published academic benchmarks
