PUBLIC NIST SP 800-22

Shadow Entropy

Zero-Cost Cryptographic Randomness from Computation

Harvests cryptographic entropy from the quotients discarded during modular arithmetic. By Landauer's Principle, every mod operation erases information—we capture it instead of throwing it away.

284 Mbit/s Rate
0 Extra Cost
~12 Bits/Reduction

How it works: In RNS (Residue Number Systems), every multiplication computes a × b mod m, discarding the quotient q = (a×b) / m. These quotients are mixed using SipHash rounds into a 256-bit accumulator, producing NIST-compliant randomness.

Applications: FHE noise sampling, key generation seeding, Monte Carlo simulations—anywhere computation already happens, entropy comes free.
